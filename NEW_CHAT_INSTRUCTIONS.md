# Инструкция для нового чата

## Как начать новый чат

1. Начните новый чат с AI ассистентом
2. Скопируйте промпт ниже
3. Дождитесь анализа файлов
4. Продолжайте работу

## Промпт для нового чата

```
Я продолжаю работу над проектом GoGoMarket.

Репозиторий: https://github.com/buranovt2025-jpg/gogomarke.uz
Ветка: devin/1767373941-gogomarket-mvp

Пожалуйста:
1. Склонируй репозиторий в /home/ubuntu/gogomarke_audit/repo/
2. Прочитай файлы:
   - QUICK_START_FOR_NEW_CHAT.md (краткий контекст)
   - CURRENT_STATUS.md (текущий статус)
   - FILES_INDEX.md (индекс всех файлов)
3. Продолжи координацию между Copilot и Cursor

Текущая задача: Backend Фаза 0 не выполнена (0/7), Frontend 5/7.
Критическая проблема: hardcoded IP в uploadService.ts блокирует Frontend.
```

## Альтернативный короткий промпт

```
Продолжаем GoGoMarket.
Repo: https://github.com/buranovt2025-jpg/gogomarke.uz
```

## Важные файлы для чтения

| Файл | Назначение |
|------|------------|
| `QUICK_START_FOR_NEW_CHAT.md` | Быстрый старт |
| `PROJECT_CONTEXT.md` | Полный контекст |
| `CURRENT_STATUS.md` | Текущий статус |
| `FILES_INDEX.md` | Индекс всех файлов |
| `COPILOT_INSTRUCTIONS.md` | Задачи для Backend |
| `CURSOR_INSTRUCTIONS.md` | Задачи для Frontend |
